System-based design studio

space design / drafting / model building

This architecture studio focused on the evolution of an element - volume, plane, and line - through modified systems. The first system spent it's adolescence in the pairing of two elements. The two elements, once defined in relation to each other, created a rule. In the following systems these two elements could only relate to each other as the rule allowed. Two elements create a component.

As the system progressed the components were broken down and then reassembled to create a more complex component. These components were - again - dictated by a series of connection rules that led to a larger, more intricate system model.

It would be this second system that dictated the language of the final design. This system would serve as an aid when creating spaces to execute the poetic philosophy of death through architecture.
